# Library Management System - Frontend

## Project Description
This is the frontend of Library Management System built using React (Vite).  
It connects with Spring Boot backend APIs to manage books, students, and issuing system.

---

## Technologies Used
- React (Vite)
- JavaScript
- HTML
- CSS
- REST API integration

---

## Features
- Student Login UI
- Book Management UI
- Issue/Return System UI
- API integration with backend
- Responsive design

---

## How to Run Project

### 1. Install dependencies
npm install

### 2. Start development server
npm run dev

### 3. Open in browser
http://localhost:5173
